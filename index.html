How to use this resource:

View the available folders by visiting this url: https://spaulmark.github.io/img/dir.json

Once you've selected a folder (for example: Cats), query the dir.json in that folder by visiting this url: https://spaulmark.github.io/img/dir.json

This json will give you a list of files you can use. For example, if you wanted to view Mewoth.png, it would be located at https://spaulmark.github.io/img/Cats/Meowth.png
