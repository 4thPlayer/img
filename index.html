How to contribute images to this resource:

Fork this repository, then make a pull request. Your images can be in any format, and should be around 100x100 pixels. Any larger and they will take too long to download. To generate/update the dir.json files, run deploy.sh.

How to use the images from this repo:

1. View the available folders by visiting this url: https://spaulmark.github.io/img/dir.json

2. Once you've selected a folder (for example: Cats), query the dir.json in that folder by visiting this url: https://spaulmark.github.io/img/Cats/dir.json

3. This json will give you a list of files you can use. For example, if you wanted to view Mewoth.png, it would be located at https://spaulmark.github.io/img/Cats/Meowth.png

Notes:

This is a service intended for machines to use, not humans. If you're a human who just wants to browse some cool images, you're best off downloading the whole repository as a zip.
